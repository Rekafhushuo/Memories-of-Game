## 图片命名规则

请将游戏封面图片放入 `covers` 文件夹，文件名格式：

```
[年份]_[序号]_游戏名.jpg 或 .png
```

例如：
- `2023_01_尼尔人工生命.jpg`
- `2023_02_霍格沃兹遗产.png`
- `2024_01_龙之信条.jpg`

## 图片规格要求

- **建议尺寸**：600 x 800 像素（3:4 比例，适合游戏封面）
- **支持格式**：jpg、jpeg、png、webp
- **文件大小**：建议不超过 2MB

## 放入图片后

1. 将所有封面图片放入 `covers` 文件夹
2. 图片会自动显示在博物馆页面中
3. 如果图片比例不是 3:4，页面会自动裁剪适配

## 获取图片建议

可以从以下途径获取游戏封面：
- Steam 商店页面
- 谷歌图片搜索 "[游戏名] cover art"
- 各大游戏媒体官网
- SteamGridDB、IGDB 等游戏数据库
